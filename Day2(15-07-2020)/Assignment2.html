Array Methods :

  1. Length() Method: Returns lenght of an Array.
  2. Sort() Method : Sorts array in acesending/decending order
  3. IndexOf() Method: Return element at specified index.
  4. Join() Method :Joins all elements of an array .
  5. Pop() & Push() Method : Pop removes last element from array .Push adds element to the array.
  6. Shift() & UnShift() Method : Shift removes first element of an array.Unshift adds element at beginning of an array.
  7. Slice() Method : Slice() method returns a new array containing a portion of the array on which it is implemented. The original remains unchanged.
  8. Splice() Method :This method is used to add/remove an item from the given array.
  9. Concate() Method : Its is used for concatenating two arrays.
  10. Map() Method :Map method creates a new array with the results of called function for every array element.
  11. Reverse() Method : This method reverses the array.
